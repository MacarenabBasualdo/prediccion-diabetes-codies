## 🇪🇸 Español

Este proyecto desarrolla un pipeline completo de Ciencia de Datos enfocado en la salud, combinando el análisis estadístico avanzado, el modelado predictivo y la visualización interactiva para identificar factores de riesgo y predecir diagnósticos de diabetes.

### 📊 Componentes del Proyecto
1. **Limpieza y Transformación de Datos (Python/Pandas):** Depuración del dataset clínico, manejo de valores nulos, inconsistencias y normalización de variables.
2. **Análisis Exploratorio de Datos - EDA:** Análisis bivariado para encontrar correlaciones clave entre variables fisiológicas y el desarrollo de la patología.
3. **Modelado Predictivo:** Implementación y evaluación de un modelo de **Regresión Logística** para estimar la probabilidad de diagnóstico basado en los factores de riesgo identificados.
4. **Dashboard Descriptivo (Power BI):** Diseño de un tablero interactivo enfocado en la descripción del contexto clínico y la distribución demográfica de la muestra.

### 🛠️ Tecnologías Utilizadas
* **Python** (Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn)
* **Power BI** (Modelado y diseño de tablero descriptivo)
* **Jupyter Notebooks**

## 🇺🇸 English

This project develops a complete healthcare-focused Data Science pipeline, combining advanced statistical analysis, predictive modeling, and interactive visualization to identify risk factors and predict diabetes diagnoses.

### 📊 Project Components
1. **Data Cleaning & Transformation (Python/Pandas):** Clinical dataset debugging, handling missing values, managing inconsistencies, and variable normalization.
2. **Exploratory Data Analysis - EDA:** Bivariate analysis to uncover key correlations between physiological variables and disease development.
3. **Predictive Modeling:** Implementation and evaluation of a **Logistic Regression** model to estimate diagnosis probability based on identified risk factors.
4. **Descriptive Dashboard (Power BI):** Design of an interactive dashboard focused on explaining the clinical context and the sample's demographic distribution.

### 🛠️ Technologies Used
* **Python** (Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn)
* **Power BI** (Data modeling and descriptive dashboard design)
* **Jupyter Notebooks**
